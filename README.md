# Lending Club 2014 Note Explorer

Shiny app that displays density distributions for various notes with respect to if they were fully paid or charged off (not paid). All notes are from January thru December 2014.

# Setup
unzip the LoanStats3c_securev1.csv.zip file. This contains a comma  seperated value(csv) file. There are two lines at the end of the csv file that must be removed before running the application.
